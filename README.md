# Easy ERP
1. 完全免费，且开源。
2. 个人的一个小兴趣，打算长期维护下去。

# 产品定位
## 使用对象
小微企业，对角色分工不是很明确，可以在公司内部局域网使用。
## 痛点
1. 目前很多进销售 2C 和 2B 的并不能在一起管理，第一阶段先完成基本地进销存能力可以同时管好 2 种模式的订单。
2. 加工型产品，如采购以千克入库，销售以盒/件出库
## 产品形式
采用单机版的模式，满足各创业公司起步使用。先各种功能完善了，后面看需求完善局域网内多角色使用。
